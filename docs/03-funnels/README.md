# 03 — Funnels

The journey from click to customer to repeat customer: **offers, landing pages, lead magnets, email sequences, upsells.** [Why funnels exist](../01-foundations/growth-marketing-map.md#why-funnels-exist-first-principles).

**The offer is the #1 lever in the entire system** ([ranking](../00-roadmap/skills-gap-analysis.md#1-offer--positioning-design-)) — so the first doc here is the most important thing you'll write in Week 1.

Two funnel archetypes you'll choose between:
- **Lead-gen / nurture** — ad → free lead magnet → email → sale. Cheap conversion event (email), good for a small learning budget. *(Default assumption for the [90-day plan](../00-roadmap/90-day-plan.md).)*
- **Direct-response / e-commerce** — ad → product page → buy → post-purchase upsell.
