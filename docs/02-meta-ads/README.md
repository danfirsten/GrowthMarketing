# 02 — Meta Ads

Campaign structure, targeting, budgets, and platform playbooks for Facebook + Instagram — the first channel ([why](../01-foundations/growth-marketing-map.md#where-meta--instagram-ads-fit)).

**The 2026 reality (read before anything else):** Meta removed granular interest targeting on **Jan 15, 2026**. Targeting is now largely automated (**Advantage+ Audience**), and **Advantage+ Sales** is the dominant campaign type. This means: keep structure *simple*, hand audiences to Meta's AI, and **compete on creative** ([creative is the lever](../04-creative/README.md)). Ignore anyone still teaching interest-stacking.

**Beginner default structure:** one campaign · broad Advantage+ Audience · 3–5 distinct creatives · optimize for your cheapest meaningful conversion event · then *leave the learning phase alone* for 3–4 days. Scale winners 10–20% every few days.
